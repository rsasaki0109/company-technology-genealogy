# company-technology-genealogy

Interactive genealogy of robotics & AI companies and their technology evolution.

ロボティクス・AI企業の技術変遷・買収・製品ラインナップをインタラクティブな系譜図で可視化します。

**[Live Demo](https://rsasaki0109.github.io/company-technology-genealogy/)**

## Categories

| Category | Companies |
|----------|-----------|
| **Autonomous Driving (L4+)** | Waymo, Cruise, Aurora, Argo AI, Zoox, Nuro, Pony.ai, WeRide |
| **ADAS / L2+** | Tesla FSD, Mobileye, Comma.ai, Momenta, Horizon Robotics |
| **AD Software & Infra** | Tier IV / Autoware, NVIDIA DRIVE, Applied Intuition |
| **Robotics Companies** | Boston Dynamics, Unitree, Agility, Figure AI, Physical Intelligence |
| **AI Labs** | OpenAI, Anthropic, Google DeepMind, Meta FAIR, xAI, Mistral, DeepSeek |
| **Sensors & Industrial** | Velodyne/Ouster, Luminar, Hesai, Kudan, FANUC, Intuitive Surgical |

## Relation Types

| Color | Type | Meaning |
|-------|------|---------|
| Green | `extends` | Technology iteration / next version |
| Cyan | `combines` | Merger / acquisition tech integration |
| Red | `replaces` | Strategic pivot / paradigm shift |
| Yellow | `inspires` | Influenced by external technology |
